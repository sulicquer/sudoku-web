# 极致数独 - Premium Retro Sudoku

一款具备极致复古美学设计、功能完备且交互流畅的现代 Web 数独游戏。

![极致数独界面](file:///Users/lijiawei/Desktop/test/screenshot.png)

## ✨ 特色功能

- **🎨 艺术级视觉**：采用复古浅色调（Retro Light）设计，融入毛玻璃滤镜与精致的阴影处理。
- **✍️ 笔记模式 (Notes)**：支持格内草稿记录（Pencil Marks），助你攻克高难度难题。
- **🔊 沉浸式音效**：基于 Web Audio API 合成的机械交互音效，提供极佳的输入反馈。
- **🎲 动态生成**：内置基于回溯算法（Backtracking）的随机关卡生成器，支持三种难度级别。
- **🛠️ 完善工具链**：
    - **撤销 (Undo)**：无限次步数撤销。
    - **提示 (Hint)**：智能填入正确数字。
    - **自动高亮**：选定格子的行列宫区域及全局相同数字高亮。
- **🎊 胜利庆典**：通关后触发全屏彩色纸屑粒子特效。

## 🛠️ 技术栈

- **HTML5**: 语意化结构。
- **Vanilla CSS**: 原生 CSS 变量、自定义动画、响应式布局。
- **JavaScript (ES6+)**: 纯原生逻辑实现，无第三方依赖。
- **Web Audio API**: 实时合成音效。

## 🚀 快速开始

### 本地运行
1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 即可开始游戏。

### 键盘快捷键
- **1-9**: 填入数字。
- **P**: 切换笔记模式。
- **Backspace / Delete**: 擦除数字。
- **方向键**: 移动选区。
